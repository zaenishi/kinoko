# KinokoKomori 🍄

**KinokoKomori** adalah proyek bot WhatsApp yang sedang saya kembangkan untuk memudahkan interaksi melalui pesan otomatis. Bot ini dirancang untuk memberikan layanan tertentu, seperti pengingat, informasi, atau hiburan, langsung melalui WhatsApp.

## 🛠️ Teknologi yang Digunakan
- **Node.js**: Untuk backend dan pengelolaan logika bot.
- **Baileys**: Untuk integrasi dengan WhatsApp.
- **MongoDB**: Untuk menyimpan data pengguna dan pengaturan bot.
